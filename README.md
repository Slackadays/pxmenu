# description

- pxmenu is a simple POSIX sh script that creates a cli menu with no external deps.

# usage

```sh
./pxmenu <file-name>
./pxmenu "string\nstring\nstring..."
```

- Do not pipe anything into de menu, since stdin could break stty mod.
